# <a name="partner-center-rest-error-codes"></a>Felkoder för Partner Center REST

**Gäller för:**

- Partnercenter
- Partner Center som drivs av 21Vianet
- Partnercenter för Microsoft Cloud Tyskland
- Välkommen till Partnercenter för Microsoft Cloud for US Government

REST-API: er för partner Center returnerar ett JSON-objekt som innehåller en status kod. Den här koden anger om din begäran lyckades eller varför den misslyckades.

## <a name="success-responses"></a>Lyckade svar

Status koden **2xx** anger att klientens begäran har tagits emot, förstått och godkänts.

## <a name="http-status-codes"></a>HTTP-statuskoder

Följande status koder för **4xx** och **5xx** indikerar ett fel:

- 400: felaktig begäran
- 401: obehörig
- 403: ej tillåtet
- 404: hittades inte
- 405: metoden tillåts inte
- 406: inte acceptabelt
- 409: konflikt, felkod
- 412: det gick inte att Precondition
- 429: för många begär Anden
- 500: internt Server fel
- 501: inte implementerat
- 502: Felaktig gateway
- 503: tjänsten är inte tillgänglig
- 504: Gateway-timeout

## <a name="error-responses"></a>Fel svar

Alla svar med status koden **4xx** eller **5xx** innehåller ett fel meddelande med ytterligare information om de fel villkor som påträffats.

I följande tabell-och kod exempel beskrivs schemat för ett fel svar:

| Namn        | Typ   | Description                                                                                                                                            |
|-------------|--------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| kod        | sträng | Returneras alltid. Anger vilken typ av fel som inträffat. Icke-null.                                                                                  |
| beskrivning | sträng | Returneras alltid. Beskriver felet i detalj och ger ytterligare felsöknings information. Icke-null, icke-tomt. Maximal längd är 1024 tecken. |
| data        | matris  | Returneras endast för vissa fel typer. En lista över fel objekt.                                                                                           |
| källa      | sträng | Returneras alltid. Orsaken till felet.                                                                                                              |

```json
{
  "code": <string>,
  "description": <string>,
  "data": [

  ],
  "source": <string>
}
WWW-Authenticate: OAuth realm=urn:cpsvc:cpid:{some cid}
```
